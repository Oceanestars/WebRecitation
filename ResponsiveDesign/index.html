Hey team! Here some examples for grid, and media query
media queries are a popular technique for delivering a tailored style sheet to desktops, laptops, tablets, and mobile phones. If you'd like to read more about media query, checkout this media query guide.For grids, I recommend you do the grid garden and read the complete guide to grid

Media Query for Mobile

        
        /* Extra small devices (phones, 600px and down) */
        @media only screen and (max-width: 600px) {
            body{
                font-size: 20px;
            }
        }

The code above says if device width is less than or equals to 600px

Media Query for Medium Size device

        
        /* Medium devices (tablets, 768px and up) */
        @media only screen and (min-width: 768px) {
            body{
                font-color: red;
            }
        }

The code above says if device width is greater than or equals to 768px

Media Query for Medium Size device

        
        /* Large devices */
        @media only screen and (min-width:1024px){ 
            body{
                font-size:16px;
            }
        }

The code above says if device width is greater than or equals to 1200px(And we also have greater than or equal to 992px)
